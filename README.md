# 평가 사이트
- jsp

## 구현기능

- 메인화면 html
- 목록 작성 데이터 db
- 등록 및 신고 모달 표현
- 로그인, 로그아웃 및 회원가입 세션
- 이메일 인증
- 평가 데이터베이스 DAO, DTO
- 검색
- 추천 및 삭제
- 기본 크로스 사이트 스크립팅 방어 (꺾쇠 치환)
